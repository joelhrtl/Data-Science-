# House Market Price Forecast Analysis
This housing dataset contains 4,548 observations and 7 variables, including information on area income, house age, number of rooms, number of bedrooms, population, price, and address. We selected this dataset because it is closely related to an important business problem: estimating housing prices. The variables in this dataset are easy to understand and have clear business meaning, which makes the dataset suitable for real estate and financial analysis.
